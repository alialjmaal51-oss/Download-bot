# 🤖 Telegram Download Bot

بوت تيليجرام لتحميل الملفات من الروابط المباشرة مع عرض شريط تقدم وسرعة التحميل.

## ✨ المميزات

- 📥 تحميل الملفات من الروابط المباشرة
- 📊 عرض شريط تقدم أزرق مع النسبة المئوية
- ⏱️ عرض الوقت المتبقي
- ⚡ عرض سرعة التحميل
- 📦 عرض الحجم الكلي والمنزل
- 🗑️ حذف الملفات تلقائياً بعد الرفع
